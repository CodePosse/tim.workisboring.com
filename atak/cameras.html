Add a camera layer to ATAK Android

  1. Open ATAK-CIV on Android.
  2. Open the main menu and choose Import Manager.
  3. Choose Import from URL.
  4. Copy one Network KML URL below and paste it into ATAK.
  5. After import, pan/zoom to California and enable the overlay if needed.
